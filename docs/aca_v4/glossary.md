# ACA v4 Glossary

- Tier 0 safety: Absolute authority safety checks (M0, M22).
- Tier 1 meta: Strategic controller authority (M3).
- Tier 2 bottleneck: Tactical resource/complexity control (M8).
- Tier 3 operational: Execution modules implementing pipeline logic.
- ACA trace: Ordered module event timeline for observability.
- Stub module: Placeholder module that emits trace and preserves state.
