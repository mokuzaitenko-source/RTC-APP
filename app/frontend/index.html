    Assistant Help

    RTC Assistant is a daily Build+Debug copilot. You describe the problem, it returns an execution plan you can run immediately.

    How To Use

    • 1. Pick Build for new work or Debug for failing behavior.
    • 2. Write one concrete request in the input (goal + constraints + deadline if known).
    • 3. Click Ask (or press Ctrl+Enter).
    • 4. Run the proposed steps, then ask one refinement for the next pass.

    What Output You Get

    • 3 immediate next actions to execute now
    • 1 verification step to confirm progress
    • 1 fallback step if the main path fails

    When Stuck

    • Switch Build/Debug mode and ask again with one clear objective.
    • Add constraints in one line: time, scope, tools, and done-criteria.
    • Ask: "Refine this with tighter checks and a safer fallback."
    • If output is still vague, ask for: "strict 3 actions + verification + fallback."
