    Assistant Help

    RTC DevX Copilot is a daily build/debug copilot. Give one concrete problem, get an execution plan you can run immediately, then refine once.

    What This App Does

    • Build: turns goals into a concrete ship plan.
    • Debug: turns failing behavior into isolate-check-fix steps.
    • Output is always operational: next actions, verification, fallback.

    60-Second Loop

    • 1. Pick Build or Debug.
    • 2. Write one concrete request: goal, constraint, deadline.
    • 3. Click Ask (or Ctrl+Enter).
    • 4. Execute step 1 immediately, then come back for one refinement.

    Output Shape

    • 3 immediate next actions to execute now
    • 1 verification step to confirm progress
    • 1 fallback step if the main path fails

    When Stuck

    • Switch Build/Debug if the plan is pointed at the wrong problem.
    • Add one line: scope limit, deadline, and done criteria.
    • Ask: "Refine this for lower risk and tighter verification checks."
    • If still vague, ask: "Give me immediate next action only."

    Advanced (Optional)

    • Press Ctrl+. to open advanced settings.
    • Use advanced settings only when you need model/risk/format/trace control.
